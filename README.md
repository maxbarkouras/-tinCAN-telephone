# tinCAN-telephone
Remotely send and receive CAN messages over WiFi.

**Hardware Requirements:**
- ESP32 (TWAI enabled)
- CAN Transceiver
  
**For OBD connections, you must also include:**
  - Buck Converter (minimum 12V at 1A)
  - OBD2 Cable
